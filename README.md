# AUTOMAÇÃO BÁSICA EM PYTHON

## UTILIZANDO GOOGLE ANTIGRAVITY PARA AUXILIAR NO PROJETO.

O intuito deste projeto é uma simulação básica de automação com base em python. O programa retira os dados de um documento (PowerBI,IBGE), retorna para um bloco de notas.


## FUNCIONALIDADES

Na pasta (Gestão de Inventário), criei 4 agentes, cada um com funcionalidades e preparos descritos. As quatro documentações dos agentes foram salvas em markdown para ter melhor desempenho e velocidade de leitrura.
